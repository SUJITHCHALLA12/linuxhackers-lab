## 📌 About

**Linux-Hackers LAB** is a beginner-friendly, AI-powered reference platform for mastering **600+ Kali Linux tools**. It combines a comprehensive tool guide with an integrated AI assistant to help learners understand ethical hacking, penetration testing commands, and core cybersecurity concepts — all in one place.

Whether you're just starting with Kali Linux or brushing up on advanced tools like Metasploit or SQLMap, this platform has you covered.

---

## ✨ Features

- 🔍 **600+ Kali Linux Tools** — Searchable reference for a wide range of pentesting tools
- 🤖 **AI Assistant** — Ask questions about any tool, technique, or command and get instant explanations
- 📋 **Wordlist Generator** — Generate custom wordlists for CTFs and security testing
- 🎯 **Beginner-Friendly** — Structured learning path for those new to ethical hacking
- ⚡ **Fast & Lightweight** — Deployed on Vercel for low-latency access worldwide
- 🌐 **No Installation Required** — Fully browser-based, no setup needed

## 🚀 Getting Started

### Using the Platform

1. Visit **[linuxhackers-lab.vercel.app](https://linuxhackers-lab.vercel.app/)**
2. Browse or search for any Kali Linux tool
3. Use the **AI Assistant** to ask questions about usage, flags, and real-world scenarios
4. Generate wordlists using the built-in **Wordlist Generator**

### Running Locally

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/linuxhackers-lab.git

# Navigate to the project directory
cd linuxhackers-lab

# Install dependencies
npm install

# Start the development server
npm run dev
```

> Open [http://localhost:3000](http://localhost:3000) in your browser.
